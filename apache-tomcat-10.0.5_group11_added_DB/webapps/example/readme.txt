Put this folder into apache-tomcat-10.0.5/webapps/ folder.
Put stopwords-en.txt into the outer apache-tomcat-10.0.5/bin/ folder.
You may visit the page through localhost:8080/example/index.html.
After you input something and press enter, you will be directed to localhost:8080/example/search.jsp.

The project/ folder is an example of how you wrap your code into a package and use it in tomcat.
